> Build a task app with React, Redux, TypeScript, Tailwind CSS.

## Installation

```shell
# Clone all project
git clone https://github.com/willy14620/tasks.git

# Install dependencies
npm install
```

## Usage

```shell
# Bundle with Webpack
npm run build

# Run on localhost
npm run start
```
